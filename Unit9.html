Unit 9 learnings

Hi colleagues

In Unit 9, a seminar on classification building an ML classifier using WEKA was presented. A simple K-Means algorithm was used and all hyperparameters interrogated. The class was shown the Within sum of squares value, and some testing using different random seed values to try and minimise the Within sum of squares value.
We also ran Expectation-maximization algorithm, which uses log likelyhood (which is a measure which represents the maximum value of the log of the probability) to validate the prediction accuracy.
