Unit 7 learnings

Hi colleagues

In Unit 7, the major focus has been feature engineering. (Duboue P, 2020) in his book The Art of Feature Engineering puts forth methods to approach structuring data using tools described for better performance of the chosen algorithm.

Some topics discussed are:
- scaling and normalisation:
Finding the max and min feature values and changing the scale to ensure fit to a given interval

- Discretization:
This is defining or finding thresholds to segment continous features into intervals/categories.

- Descriptive features,
This method for feature engineering is very interesting, in it, we see how data can be transformed from the raw actual data to histograms for example, where we can see the shape of the data distribution and use this for input to the algorithm.

- Outlier detection:
(Duboue P, 2020) makes an argument for not discarding outliers, unless you are very sure these are in fact errors. A model is more robust if all data, even extreme cases are included.
